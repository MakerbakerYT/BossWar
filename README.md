# BossWar
> 보스전 플러그인

## 사용법
* [x] /보스스폰 - 보스스폰위치 설정
* [ ] /보스입장 - 보스전 입장

## 기능
서버가 열린 후 10분 뒤, 보스가 죽은 뒤 20분 때 보스가 스폰됩니다.

### 스킬 안내
> 폭발
>> 5칸 이내의 플레이어에게 데미지를 입힙니다.

> 번개
>> 5칸 이내의 플레이어 중 3명을 뽑아서 즉사
